# Definition

Dropdown is a clickable interactive element that reveals a Card component with any children you provide.

You can place text, icons, or other elements inside a Dropdown.

Dropdowns are often used for navigation, selection menus, and filtering content.

## Usage

This customizable component features adjustable elements:

- `dropdownContent` - content to reveal
